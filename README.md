# Tic Tac Toe

Welcome to the Tic Tac Toe game, This application allows you to play the classic game of Tic Tac Toe on a single device.

## Introduction

Tic Tac Toe is a simple yet fun game for two players, X and O, who take turns marking spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Features

Two-Player Mode: Play against a friend on the same device.

User-Friendly Interface: Easy to use and navigate.

Reset Functionality: Restart the game at any time.
